# Mini Blockchain Implementation

This is a Java implementation of a mini blockchain system as specified in the COMP4137/COMP7200 Programming Project.

## Features

1. Transaction generation with ECDSA signatures
2. Verifiable Merkle tree of transactions
3. Construction of blockchain with Proof-of-Work mining
4. Integrity verification of transactions and blocks

## Requirements

- Java 11 or higher
- Maven 3.6.0 or higher

## Dependencies

- Bouncy Castle (for ECDSA cryptography)
